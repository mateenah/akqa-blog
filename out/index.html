Re Think Plastic

Without plastic, there is not much we could do today. Plastic is in everything like packaging, furniture, cars, electronics. So we know it's an important resource. Or do we really do that? Would we not handle plastic in a better way if we did it? Plastic is made of finite resources. But much of these resources are burned only today and the value goes up in smoke. We need to think differently. It is high time to re-evaluate the plastic.
